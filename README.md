# ささっと算数アプリ。学ぼう

小学生の子どもたちが楽しく遊びながら算数を学べるWebアプリ集です。

## 🎮 ゲーム一覧

### 1. パーセントマスター！お買い物クエスト
割合とパーセントをマスターするゲーム。お買い物をしながら、レベルアップでどんどん難しくなっていきます。

**特徴:**
- レベル1から10まで段階的に学習
- 視覚的なタイル図と数直線図で理解を深める
- 間違えた時の詳しい解説でしっかり学べる
- バッジコレクションで達成感を味わえる
- 連続正解でコンボボーナスがもらえる
- 手書き筆算コーナーで計算の練習もできる

### 2. 公倍数パニック！キラキラ星あつめ
流れてくる星をタッチして、2つの数の公倍数を見つけるゲーム。スピード感あふれるゲームで、公倍数の計算が得意になります。

**特徴:**
- 動きのある星をタッチして楽しく学習
- 公倍数の概念を直感的に理解できる
- スコアを競ってモチベーションアップ
- 時間制限で集中力も鍛えられる

### 3. もぐもぐ約分マスター
かわいいクマと一緒に、分数の約分をマスターするゲーム。レベルアップで音楽や背景が変わって、楽しく繰り返し学習できます。

**特徴:**
- レベル1から10まで段階的に学習
- バッジコレクションで成長を実感
- タイムアタックモードでスピードも鍛えられる
- かわいいキャラクターと一緒に楽しく学習
- レベルアップで音楽や背景が変わる

## 🚀 使い方

1. [index.html](index.html) を開いて、遊びたいゲームを選びます
2. 各ゲームの「🎮 あそぶ」ボタンをクリックして開始します
